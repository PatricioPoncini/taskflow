# `TaskFlow`
A small backend system for managing projects and tasks

## `Tech stack`
[![My Skills](https://skillicons.dev/icons?i=ts,nodejs,express,mongodb,npm)](https://skillicons.dev)

## `Setup the project`
1. Clone the repository
2. Navigate to the project directory
3. Install dependencies by running `npm i`
4. Rename `.env.example` file to `.env` and fill in the required variables
5. Start the project locally by running `npm run dev`
